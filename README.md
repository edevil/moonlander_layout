# moonlander_layout

Portuguese Mac keyboard layout for ZSA Moonlander

# Building

1. Install QMK cli:

        brew install qmk/qmk/qmk
        qmk setup

1. Use QMK (ZSA version firmware24)
1. Create link ./keyboards/zsa/moonlander/keymaps/edevil to this repo

1. Compile

        make zsa/moonlander:edevil
